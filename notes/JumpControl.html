Jump control bits

Determines whether the program counter should jump to the address in the address register or continue with the next instruction.

The decision is based on the following control bits and the output from the ALU.

conditionmeaning
j1j2j3
< 0== 0> 0
000don't jump
001jump if output > 0
010jump if output == 0
011jump if output >= 0
100jump if output < 0
101jump if output != 0
110jump if output <= 0
111always jump
