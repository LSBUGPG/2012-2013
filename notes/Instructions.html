Load instruction

Loads the given value into the address register. The range of this value is 0 to 32,767. Bit 16 must be 0 otherwise the instruction will be interpreted as a compute instruction. Hex digit 4 therefore cannot be greater than 7.

Compute instruction

Performs a computation on the ALU:

  • a - if set, the value from RAM is passed to the ALU
  • c1 to c6 - these program the ALU
  • d1 - if set, loads output into address register
  • d2 - if set, loads output into data register
  • d3 - if set, loads output into RAM
  • j1 to j3 - program the jump control
